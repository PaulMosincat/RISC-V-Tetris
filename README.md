# RISC-V-Tetris

A Tetris game developed in RISC-V assembly language as part of a digital systems and computer architecture project.

## Overview
This project focuses on implementing core Tetris mechanics using low-level programming concepts in assembly language. The game includes piece movement, collision handling, row clearing, and keyboard controls.

## Features
- Piece movement and rotation
- Collision detection
- Row clearing logic
- Game loop implementation
- Keyboard input handling

## Technologies Used
- RISC-V Assembly
- RARS Simulator

## What I Learned
- Low-level systems programming
- Register and memory management
- Debugging assembly code
- Implementing game logic in assembly
- Problem solving with constrained system resources
